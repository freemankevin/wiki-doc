# 资源限制

> Docker 相关的 资源限制 问题收集
{.is-info}

---

```yaml
version: '3.8'

services:
  web:
    image: nginx:latest
    container_name: web
    ports:
      - "80:80"
    deploy:
      resources:
        limits:
          cpus: '1.5'            # 限制 CPU 为 1.5 核
          memory: 512M           # 限制内存为 512MB
        reservations:
          cpus: '1'              # 预留 1 核 CPU
          memory: 256M           # 预留 256MB 内存
```
